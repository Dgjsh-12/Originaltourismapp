こちらのWebアプリは観光スポットとなっております。  
行きたい観光スポットをエリア一覧で選択出来ますし観光スポットの情報や他のユーザーが観光に行った時の感想なのが投稿されており見られる事が出来ます。自分でも観光に行った時の感想を投稿する事も出来ます。投稿したのを編集、削除と言った事も出来ます。
